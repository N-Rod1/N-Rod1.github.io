My Website

I found the bootstrap template on Bootstrapmade.com but all the editing was done by myself. I started with getting rid of any code I was positive I would not need, which turned out to be a lot. The template was massive. I then started changing the background image and all other images. I changed the menu and made several pages for what I needed to make the website my own. I have five menu items one is just a home button, but the other four have all the main content of the website. I spent a lot of time learning how all the pages link together and formatting the CSS pages to what I needed to make the website look good. I'm really proud of all that I learned in this class to make this possible and I look forward to what I will learn next. Thank-you Professor Rio for guiding me through this and teaching me.
